# python-starting
初期環境向けPythonパッケージパックです。

## How to use
必要な環境にrequirements.txtをダウンロードして、以下のコマンドを実行するだけ！
```console
pip install -r requirements.txt
```
ダウンロードは以下のコマンドでこのリポジトリ全体をダウンロードしていらないファイルを消すだけ
```console
git clone https://github.com/Seki-Satoshi/python-starting.git
```
もしくはこちらからどうぞ
```console
curl -O [https://github.com/Seki-Satoshi/python-starting/blob/1dbcf43d65bff7e00d5b3d3caba1aa20401215f9/requirements.txt](https://raw.githubusercontent.com/Seki-Satoshi/python-starting/1dbcf43d65bff7e00d5b3d3caba1aa20401215f9/requirements.txt)
```
